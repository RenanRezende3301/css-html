Testing HTML5 Tags

Most Common Tags

The tag: strong

This tag is used to emphasize a part of the text, typically used to indicate something important. The browser applies a bold style by default.

The tag: em

This tag is used to emphasize a part of the text. Unlike strong, the emphasis is more subtle and the browser applies an italic style by default.

The tag: u

This tag adds underline to the text. It can be used to highlight important information or for specific cases, such as links in printed texts.

The tag: mark

This tag is used to highlight important parts of the text, applying a yellow background or another style configured by the browser.

The tag: del

This tag is used to indicate text that has been removed or is no longer valid. The browser applies a strikethrough style by default.

The tag: ins

This tag indicates text that has been added or changed, usually highlighted with an underline. It is useful for revisions and content modifications.

The tag: small

This tag reduces the text size. It is often used for footnotes or secondary comments. This is an example of smaller text.

The tag: code

This tag styles text as programming code, usually with a monospaced font. Ideal for highlighting code snippets like console.log("Hello, world!");.

The tag:
blockquote

This tag is used to insert a long quotation. The browser typically applies an indentation to the text:

"This is a sample quote using the blockquote tag."

The tag: a

This tag creates a link to another page or resource. It is used in conjunction with the href attribute, which specifies the link destination. Click here for an example.
